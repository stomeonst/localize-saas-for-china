## Safety design

The skill will not invent customers, integrations, certifications, local hosting, Chinese support, response times, pricing, regulatory status, or performance results.

It separates verified facts, user-provided facts awaiting verification, inferences, and missing evidence. Customer-facing copy uses verified facts only. It also treats mainland accessibility, data and security documentation, and the procurement path as default launch gates until current evidence resolves them.

The skill prepares drafts. It does not publish, send messages, register accounts, purchase services, request credentials, or handle payments.
